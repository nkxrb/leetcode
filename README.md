# leetcode

记录leetcode上的一些题解等等

使用Jest来测试程序是否通过，使用IDE编码，更好的编码体验，更好的本地调试
如果可以给你带来更好的刷题体验，欢迎fork，star（^.^）

### 技术栈
- JavaScript
- TypeScript
- Jest + babel
- ES6

### 目录结构
- \__test__  测试文件
- problems 题解文件
- utils 一些工具类，封装的二叉树、链表等结构

### 使用方式

安装依赖
```bash
pnpm i
```

测试单个题解, 例如测试35.test.js，其中35为需要测试的题号
```bash
npm run test 35
```

测试所有示例
```bash
npm run test
```
